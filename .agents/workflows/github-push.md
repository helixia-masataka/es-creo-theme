---
description: 変更内容をGitHubに追加・反映（プッシュ）する
---

テーマの編集内容などを、ご自身のGitHubの main ブランチにプッシュする手順です。
（実行する際はコミットメッセージの内容を適宜書き換えてください！）

1. すべての変更をステージングする
```bash
git add .
```

2. 変更をコミットとしてローカルに保存する（セーブポイントを作る）
```bash
git commit -m "Update: テーマのスタイルと機能などを修正"
```

3. リモートリポジトリ（GitHub）へプッシュする
```bash
git push origin main
```
